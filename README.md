# featurevisor-example-ruby

Example Ruby application that demonstrates how to use Featurevisor [Ruby SDK](https://github.com/featurevisor/featurevisor-ruby).

Learn more about Featurevisor [here](https://featurevisor.com).

## Installation

Clone the repository, and run:

```
$ bundle install
```

## Usage

Run the server locally, running:

```
$ rake run
```

## Featurevisor project

Uses this Featurevisor project to fetch the configuration from: https://github.com/featurevisor/featurevisor-example-cloudflare
